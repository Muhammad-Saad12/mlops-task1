This is a simple task for MLOPs class.
To run this project, you need to have a working environment with the following libraries installed:
-python
-git

Follow the steps below to run the project:
1. pull the repo by running the command below:
    git pull https://github.com/Muhammad-Saad12/mlops-task1
2. To run this project run the following command:
    python task1.py
3. Make changes to the code
4. Push the changes to the repo by running the command below:
    git add .
    git commit -m "your message"
    git push -u origin dev
